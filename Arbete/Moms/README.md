# Arbete/Moms/

Momssammanställningar och kontrollberäkningar per redovisningsperiod, sorterade i undermappar
(t.ex. `Q2-2026/`). Underlag stäms av mot Fortnox momsrapport inför inlämning till Skatteverket.

**Säkerhet:** Skarpa filer här innehåller verkliga siffror och kan innehålla personuppgifter — de är
**gitignorerade**. Endast denna README spåras i git. Se projektets `.gitignore`.
