# Arbete/

Här ligger faktiska exporter och genererade underlag — separat från kunskapskällorna i `Knowledge Base/`.
Innehållet kan vara känsligt (bolagets siffror). Dela inte utan eftertanke; exponera inga secrets.

- `Fortnox-exporter/` — råa rapporter exporterade från Fortnox (Resultatrapport, Balansrapport, Huvudbok, Momsrapport, kontoutdrag m.m.), sorterade per räkenskapsår/period.
- `Årsredovisning/` — arbetsversioner och sammanställningar för årsredovisningen RÅ 2024-11-28–2025-12-31.
- `Inkomstdeklaration2/` — INK2R/INK2S-underlag och skatteberäkning.
- `Moms/` — momssammanställningar per period (t.ex. `Q2-2026/`).
