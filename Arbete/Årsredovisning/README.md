# Arbete/Årsredovisning/

Genererade arbetsfiler för bokslut och årsredovisning (utkast, avstämningar, stämmoprotokoll,
fastställelseintyg).

**Säkerhet:** Skarpa filer här innehåller verkliga siffror och kan innehålla personuppgifter — de är
**gitignorerade**. Endast denna README spåras i git. Se projektets `.gitignore`.
