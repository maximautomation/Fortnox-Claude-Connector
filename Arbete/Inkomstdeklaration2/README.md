# Arbete/Inkomstdeklaration2/

Genererade arbetsfiler för Inkomstdeklaration 2 (räkenskapsschema INK2R, skattemässiga
justeringar INK2S, bolagsskatteberäkning och SRU-underlag). Bygger på det färdiga
bokslutet/årsredovisningen.

**Säkerhet:** Skarpa filer här innehåller verkliga siffror och kan innehålla personuppgifter — de är
**gitignorerade**. Endast denna README spåras i git. Se projektets `.gitignore`.
