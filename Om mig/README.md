# Om mig/ — bolagsspecifika fakta (KÄNSLIGT)

Denna mapp innehåller **ditt bolags verkliga uppgifter** och behandlas som känslig. Allt utom
mallar och denna README är **gitignorerat** — det laddas aldrig upp till GitHub.

## Vad som ska ligga här (ditt eget)
- `Bolagsfakta.md` — sammanfattning som CLAUDE.md läser först (kopiera från `EXEMPEL-Bolagsfakta.md`).
- Bolagsordning, registreringsbevis/registerutdrag, stiftelseurkund.
- Myndighetsbeslut (t.ex. momsbeslut), samt övriga bolagsdokument.

## Spårat i git (delbart)
- `README.md` (denna fil)
- `EXEMPEL-Bolagsfakta.md` (mall utan riktiga uppgifter)

## Säkerhet
Innehåller personnummer och bolagsuppgifter. Ladda aldrig upp innehållet publikt. Se projektets
`.gitignore` och `README.md` för säkerhetsmodellen.
