Ingen kan skapa morgondagens lösningar helt själv, vi gör det bäst tillsammans.

## Kort om träffarna

### Varför

Vi vill ge er bättre förutsättningar att skapa upplevelser som förenklar vardagen för företagare. Till exempel kan man numera som programvaruföretag vara ombud för ett företag och ta del av information på skattekonton. Vad mer skulle vi kunna göra och varför? Vi vill att ni är en del av lösningen.

De här träffarna syftar till att vi aktivt ska kunna diskutera och reflektera kring vilka utmaningar som finns hos er och vad vi tillsammans kan göra för att lösa dem. Det kan handla om små förbättringar i närtid eller stora förflyttningar som behövs i framtiden. Vi välkomnar stora som små förslag, färdiga som ofärdiga.

Vi publicerar även information om kommande träffar som nyheter.

- [Nyheter om digitala samarbeten](https://www.skatteverket.se/omoss/digitalasamarbeten/prenumererapanyheterviarss.4.1657ce2817f5a993c3a5403.html)

### Hur

Vi kommer använda videolänk, mer information kommer på mejl inför mötet.

### Så anmäler du ditt intresse

Fyll i vårt formulär för samarbeten om du tror att det här är något för dig och dina kollegor. Har du inte varit med tidigare kommer vi att kontakta dig innan mötet och berätta mer om förutsättningarna.

- [Kontakta oss för samarbeten](https://www.skatteverket.se/omoss/digitalasamarbeten/samarbeten/kontaktaossforsamarbeten.4.7c708f0e16bed42cd055c16.html)

### Bra att veta inför mötet

Det finns ett begränsat antal platser, först till kvarn gäller.
