Öppna data och öppna API:er är information som finns tillgänglig för vem som helst att använda, återanvända och dela med sig av. Skatteverkets öppna data och API:er är alltid en sammanställning av många olika data, ofta baserat på år eller del av ett år. Det går inte att få ut uppgifter på individnivå med öppen data.

## Skatteverkets öppna data och öppna API:er

Information som vi som myndighet har kan ofta användas utanför den offentliga förvaltningen med syfte att skapa kommersiella eller ideella tjänster. Den typen av vidareutnyttjande av myndighetsinformation regleras i PSI-lagen. PSI står för Public Sector Information.

På Skatteverket finns en stor mängd öppna data och öppna API:er som alla kan ta del av. Vi publicerar löpande nya eller uppdaterade datamängder.

- [Utvecklarportalen Länk till annan webbplats.](https://www7.skatteverket.se/portal/apier-och-oppna-data/utvecklarportalen?dataresurs=oppna-data&dataresurs=oppna-data-api)

Om du vill veta mer om vad öppna data är för något så föreslår vi att du börjar här:

- [Vad är öppna data?](https://www.skatteverket.se/omoss/digitalasamarbeten/omvaraoppnadata/vadaroppnadata.4.96cca41179bad4b1aa68ec.html)

### Exempelformat på våra dataresurser

Du kan komma åt Skatteverkets öppna data på flera sätt. Många öppna data finns både som exempelvis CSV-fil och öppet API, ibland finns dock bara ett format. Här är några varianter:

- XLSX
- CSV
- Excel
- API

## Personuppgifter lämnas aldrig ut som öppen data

Folkbokföringsuppgifter eller uppgifter om inkomster och skatt finns inte som öppna data och går därför inte att få via API.

All data på individnivå är skyddad av särskilda regler och kan inte lämnas ut utan medgivande från den det berör. Idag finns inga bra och enkla lösningar för ett sådant medgivande, och därför finns inte heller någon tillgång till individdata för tredje part.

Du kan läsa mer om vilka regler som gäller för vilka uppgifter du kan begära ut om dig själv och andra på sidan om offentlighet och sekretess.

- [Offentliga och sekretessbelagda uppgifter](https://www.skatteverket.se/omoss/varverksamhet/offentligauppgifter.4.5c281c7015abecc2e20333e1.html)
