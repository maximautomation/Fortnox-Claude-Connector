API:et riktar sig till offentliga aktörer och programvaruföretag som utvecklar tjänster för att visa ärendeåterkoppling i form av kundhändelser. Med API:et kan du hämta kundhändelser från flera olika myndigheter och kommuner samtidigt, inte bara från Skatteverket.

## Fyll i formuläret

Fyll i och skicka in formuläret om du önskar få tillgång till API Mina ärenden kundhändelser. API:et finns tillgängligt både som komplett testtjänst och driftsatt tjänst.

Läs mer om API:ets teknik, operationer och tjänstebeskrivning på Utvecklarportalen.

- [API Mina ärenden kundhändelser Länk till annan webbplats.](https://www7.skatteverket.se/portal/apier-och-oppna-data/utvecklarportalen?q=kundh%C3%A4ndelser)

Fyll gärna i en gemensam mejladress (så kallad funktionsbrevlåda) i fältet 'Mejladress till kontaktperson'.

## Om API:et

API:et finns tillgängligt både som komplett testtjänst och driftsatt tjänst.

Läs mer om API:ets teknik, operationer och tjänstebeskrivning på Utvecklarportalen.
