Skatteverket har många olika registeruppgifter för alla de företag som finns eller verkar i Sverige, till exempel om ett företag har F-skattsedel eller om det är moms- och arbetsgivarregistrerat. För att främja en enklare vardag för företagare utvecklar vi just nu ett API där programvaruföretag kan hämta företagsinformation och visa den direkt i den egna programvaran.

## Kort om utvecklingsområdet

Skatteverkets registeruppgifterna kan användas i en mängd olika situationer, både av företaget själv och av andra. För egen del kan företag exempelvis använda uppgifterna i olika ansökningstjänster eller om de ska registreras in i ett nytt affärssystem. Andra kan använda det i KYC-syfte (Know Your Customer), exempelvis som underlag vid en upphandling eller när du ska anlita ett företag för Rot- och Rut-arbeten.

Vi vill hitta scenarion där vi kan tillgängliggöra de här uppgifterna på ett säkert sätt som gynnar de inblandade parterna i situationen, och som även främjar en enklare vardag för företagare i allmänhet.

## Färdiga tjänster finns på Utvecklarportalen

Inom området för företagsuppgifter kan du till exempel använda API:et Beskattningsengagemang.

- [Utvecklarportalen Länk till annan webbplats.](https://www7.skatteverket.se/portal/apier-och-oppna-data/utvecklarportalen?q=beskattningsengagemang)

## Pågående utveckling

### Hämta företagsinformation

Skatteverket erbjuder idag en tjänst på Skatteverket.se där företag och privatpersoner kan söka på företag om de exempelvis är godkända för F-skatt, registrerade för moms eller registrerade som arbetsgivare:

- [Hämta företagsinformation Länk till annan webbplats.](https://www.skatteverket.se/privat/etjansterochblanketter/allaetjanster/tjanster/hamtaforetagsinformation.4.3810a01c150939e893f3e69.html?q=h%C3%A4mta+f%C3%B6retagsuppgifter)

Informationen kan användas som till exempel underlag vid en upphandling eller när du ska anlita ett företag för Rot- och Rut-arbeten.

Tjänsten kräver idag manuell inmatning av information där svaret ges via e-post. Skatteverket har startat en utredning för att se om det finns legala och tekniska möjligheter att erbjuda denna tjänst där ett mer maskinellt informationsutbyte kan ske via API. Det skulle bland annat kunna göra att det blir enklare som en ny kund i ett bokförings-/affärssystem att få in rätt uppgifter/förutsättningar automatiskt, eller motsvarande för ombud som får nya kunder de ska hjälpa.

Det finns just nu inga planer på ett API för att Hämta företagsinformation.
