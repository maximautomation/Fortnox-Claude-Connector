Förutom öppna data och API:er erbjuder Skatteverket även statistik. Statistiken hittar du på Statistikportalen. Du som har behov av att hämta statistiken med hjälp av API kan göra det på Sveriges dataportal.

På Skatteverket skiljer vi på öppna data och statistik, men båda typerna av data är fria att använda för dig som vill ta del av och använda den, till valfritt ändamål. Statistik baseras ofta på öppen data, men är mer bearbetad.

Våra öppna data hittar du på Utvecklarportalen. Det kan vara om allt från medelinkomster i olika branscher till genomsnittshyror för butikslokaler och annat.

Statistik om det svenska skattesystemet, samt namn på nyfödda, hittar du istället på Statistikportalen. Du kan ladda ner statistiken i olika format och den finns också tillgänglig som tabeller och diagram.

- [Utvecklarportalen Länk till annan webbplats.](https://www7.skatteverket.se/portal/apier-och-oppna-data/utvecklarportalen)
- [Statistikportalen](https://www.skatteverket.se/omoss/varverksamhet/statistikportalen.4.6704c7931254eefbe718000121.html)

## Statistik, öppna data och API:er på Sveriges dataportal

För dig som har behov av automatisk hämtning av statistik finns den även som API på Sveriges dataportal. Du får tips på hur du gör för att hämta statistiken under fliken Hämta statistik med API på Statistikportalen.

På Sveriges dataportal hittar du även de öppna data-samlingar samt API:er som finns på Utvecklarportalen. Sveriges dataportal samlar öppna och delade data från svenska myndigheter, regioner och kommuner.

- [Sveriges dataportal för ökad innovationskraft Länk till annan webbplats.](https://www.dataportal.se/)

## Viktigt om sekretess

Både öppna data och statistik från Skatteverket består av aggregerad data. Det går alltså inte att få ut öppna data eller statistik där privatpersoner eller företag kan identifieras, direkt eller indirekt. Det beror på att elektronisk utlämning av data på individnivå är reglerat av särskilda lagar.
