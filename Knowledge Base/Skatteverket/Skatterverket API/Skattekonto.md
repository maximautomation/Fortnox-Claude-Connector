Skattekontot berör alla företag som bedriver näringsverksamhet. Kontot visar bland annat företagets skatteinbetalningar, preliminära skatt och slutlig skatt. Skatteverket erbjuder i dagsläget ett API inom området.

## Kort om utvecklingsområdet

Ett företag bevakar ofta flera transaktioner på skattekontot varje månad. Skatter och avgifter som ska betalas stäms av mot de inbetalningar som gjorts och vid de månatliga saldoavstämningarna registreras eventuell ränta.

Allt vi utvecklar inom det här området syftar därför till att underlätta för slutkonsumenter som har behov av löpande bokföringsarbete och skattekontoavstämningar. Du som programvaruföretag kan därmed skapa automatiserade processer som minskar risken för manuella fel.

## Det här gör vi inom området

### Befintliga API:er och API:er under utveckling

I dagsläget finns ett API som heter Skattekonto. Det består just nu av fyra operationer:

- hämta anmält utbetalningskonto
- hämta aktuellt dagssaldo
- hämta tidigare transaktioner (två och ett halvt år bakåt i tiden) och kommande transaktioner
- hämta utbetalningsspärr gjord på egen begäran

Du hittar alla detaljer på Skatteverkets utvecklarportal.

- [Skattekonto Länk till annan webbplats.](https://www7.skatteverket.se/portal/apier-och-oppna-data/utvecklarportalen?q=skattekonto&dataresurs=partner-api)

Du kan även begära tillgång till komplett testtjänst och driftsatt tjänst här.

- [Begär tillgång till våra API:er](https://www.skatteverket.se/omoss/digitalasamarbeten/anslutningochsupportforvaraapierochoppnadata/begartillgangtillvaraapier.4.109dcbe71721adafd256d8f.html)

## Utvecklad läsbehörighet skapar fler möjligheter

Nu kan även en organisation få läsbehörighet för skattekontot i tjänsten Ombud och behörigheter. Organisationen kan legitimera sig via sitt system med sin organisationslegitimation för att få tillgång till skattekontouppgifterna i API:et. Den nya läsbehörigheten finns tillgänglig i komplett testtjänst och driftsatt tjänst.
