I formatet öppna data gör Skatteverket information tillgänglig för vem som helst att använda, återanvända och dela med sig av. Genom att andra än Skatteverket kan använda vår data i sin utveckling skapas nytta för fler.

Vi har endast öppna data på aggregerad nivå, alltså en sammanställning av många olika data. Det går inte att få ut data på individnivå via våra öppna data. Du kan exempelvis inte få ut adress- eller inkomstutgifter för privatpersoner digitalt via öppna data.

Vår öppna data är fri att använda och kräver varken några avtal eller innebär några avgifter.

## Format för öppna data

Informationen lämnas i ett format som följer en öppen standard, främst csv (comma-separated values) och ett tillhörande API (application programming interface). För informationsmängder där andra format efterfrågats försöker vi tillhandahålla även dessa. Det kan vara till exempel excel eller txt.

Vi publicerar informationen i så maskinläsbar form som möjligt.

## Lagar om att använda öppna data

Vidareutnyttjande av myndighetsinformation regleras i PSI-lagen. PSI står för "Public Sector Information", alltså offentlig information.

- [Lag (2022:818) om den offentliga sektorns tillgängliggörande av data Länk till annan webbplats.](https://www.riksdagen.se/sv/dokument-och-lagar/dokument/svensk-forfattningssamling/lag-2022818-om-den-offentliga-sektorns_sfs-2022-818/)

Information för vidareutnyttjande omfattar aldrig information där privatpersoner eller företag kan identifieras, varken direkt eller indirekt. Det regleras i dessa lagar:

- [EU:s dataskyddsförordning Länk till annan webbplats.](https://eur-lex.europa.eu/legal-content/SV/TXT/?qid=1525690211521&uri=CELEX:32016R0679)
- [Lag (2001:181) om behandling av uppgifter i Skatteverkets beskattningsverksamhet Länk till annan webbplats.](https://www.riksdagen.se/sv/dokument-och-lagar/dokument/svensk-forfattningssamling/lag-2001181-om-behandling-av-uppgifter-i_sfs-2001-181/)

## Vad kan öppna data användas till?

Det finns inga egentliga begränsningar för vad öppna data får eller kan användas till. Grundtanken är att den är fri att använda av vem som helst, till vad som helst. Öppna data kan bland annat användas av företag, studenter och forskare, journalister eller privatpersoner som är intresserade av datan.

#### Exempel 1

En journalist begär, via ett öppet API från Skatteverket, ut alla nybilspriser från 2011 till 2021. Prislistan skickas i form av en datatabell till ett kalkylark på journalistens dator, som nu har ett underlag till en artikel om bilprisernas utveckling under 10 års tid.

#### Exempel 2

En programutvecklare vill presentera hur flygskatten påverkas av väder och geografi. Hens idé är att slå ihop kartdata från Lantmäteriet, väderdata från SMHI och data om flygskatt från Skatteverket.

Utvecklaren anropar sedan API:er från respektive myndighet och kombinerar sedan all data i en och samma app, där användarna kan utforska hur flygskatten påverkas av väder och geografi.

Läs gärna vidare i artikeln:

- [Vad är öppna data?](https://www.skatteverket.se/omoss/digitalasamarbeten/omvaraoppnadata/vadaroppnadata.4.96cca41179bad4b1aa68ec.html)

## Offentliga uppgifter från andra aktörer

För den som uppfyller vissa villkor kan Statens person­adress­register, SPAR, användas för att få ut uppgifter på individnivå. Det är ett offentligt register som omfattar alla personer som är folkbokförda i Sverige, både svenska och utländska medborgare.

- [SPAR - Statens personadressregister Länk till annan webbplats.](https://www.statenspersonadressregister.se/master/start/)
