En inkomstdeklaration är ett formulär där fysiska och juridiska personer redovisar uppgifter om inkomster för ett beskattningsår. Baserat på den inskickade inkomstdeklarationen räknar Skatteverket ut årets slutliga skatter och avgifter. Vi har API:er som gör att du både kan hämta förifyllda uppgifter i deklarationen och skicka in deklarationen.

## Kort om utvecklingsområdet

Det finns fyra olika inkomstdeklarationer, Inkomstdeklaration 1, 2, 3 och 4. Många företag, föreningar och organisationer tar hjälp av en bokförings- eller redovisningsbyrå för att upprätta deklarationen. Om byrån är utsedd till ombud för företaget kan den utföra ärenden hos Skatteverket för företagets räkning, till exempel skriva under deklarationen. När deklarationen upprättas används ofta ett skatteprogram.

Vi har ett API som omfattar Inkomstdeklaration 2, 3 och 4 och det finns som både komplett testtjänst och driftsatt tjänst. Med API:et kan du hämta förifyllda uppgifter i deklarationen och skicka in deklarationen.

En förenklad inlämning av Inkomstdeklaration 2, 3 och 4 i driftsatt tjänst ser vi som viktig då många företagare har stöd av en programvara när de upprättar deklarationen. Utveckling inom området gör det också lättare att göra rätt, inte bara för företagen, utan även för deras byråer.

## Under utveckling

Utveckling pågår av en testtjänst för API Inkomstdeklaration 1, med preliminär lansering under våren 2026.

## API Inkomstdeklaration 2-4

Med API Inkomstdeklaration 2-4 kan företag, föreningar, organisationer eller ombud se den förifyllda inkomstdeklarationen direkt i sin programvara. När deklarationen är upprättad och klar går det att skicka in inkomstdeklarationen för underskrift direkt från programvaran. Behörig firmatecknare eller deklarationsombud skriver under inkomstdeklarationen på Skatteverkets Mina sidor.

API:et innehåller följande operationer:

- Hämta deklarationsstatus för respektive beskattningsår
- Hämta förifyllda uppgifter i Inkomstdeklaration 2, 3 och 4
- Skicka in deklarationsunderlag för Inkomstdeklaration 2, 3 och 4

API:et finns som komplett testtjänst och driftsatt tjänst.
