Skatteverket vill göra det möjligt för företag och privatpersoner att ta del av skatteregler direkt i sina egna miljöer, som appar och affärssystem. Vi erbjuder ett API och regelfiler som kan användas tillsammans med en regelmotor.

## Kort om utvecklingsområdet

De flesta företagare hanterar skatteregler i sina affärssystem. Skatteregler ändras ofta, vilket nästan alltid kräver manuell hantering och uppdatering i systemen. Detta komplexa område ställer krav både på Skatteverket och på dig som utvecklar programvaror.

Vi tror att det skulle bli lättare för programvaruföretagen att underhålla reglerna om aktuella versioner av skattelagstiftningen fanns tillgängliga direkt i deras affärssystem. Det skulle också leda till ökade möjligheter till automation, samt höjd kvalitet i redovisning och uppgiftslämnande.

Arbetet med att göra lagstiftning möjlig att digitalisera ingår i OECD:s vision "Rules as code".

## Använd vårt API om du hanterar skatteregler

Med API:et Rättsliga regler kan programvaruutvecklare hämta maskinläsbara regelfiler med Skatteverkets tolkning av skatteregler inom ett område. Du behöver teckna avtal med Skatteverket för att få använda API:et. Regelfilerna finns som öppen data och är tillgängliga för alla. Alla befintliga regelfiler finns i JSON- och BPMN-format. Nya regelfiler publiceras i BPMN-format.

På vår Utvecklarportal hittar du beskrivning av API:et, samt de regelfiler som finns tillgängliga. Vår guide ”Så använder du regelfiler och regelmotor” hjälper dig att komma igång. Guiden kommer att uppdateras med tips även för BPMN-filerna.

- [API:et Rättsliga regler (Utvecklarportalen) Länk till annan webbplats.](https://www7.skatteverket.se/portal/apier-och-oppna-data/utvecklarportalen?q=R%C3%A4ttsliga%20regler&dataresurs=partner-api)
- [Rättsliga regelfiler (Utvecklarportalen) Länk till annan webbplats.](https://www7.skatteverket.se/portal/apier-och-oppna-data/utvecklarportalen/oppetdata/R%C3%A4ttsliga%20regelfiler)

## Pågående utveckling

Vi siktar på att publicera ett nytt regelområde i början av sommaren och utökar regelbundet utbudet av regelfiler.

Skatteverket utvärderar även om generativ AI kan användas i arbetet med att ta fram regelfiler och om regelfiler i sin tur kan förbättra AI-svaren.

## Kom gärna med synpunkter och förbättringsförslag

Vi tar tacksamt emot synpunkter på funktionaliteten i vårt API, regelfilernas innehåll och uppbyggnad samt utbudet av regelfiler. Det finns mycket att utforska kring den här tekniken och vårt arbete befinner sig fortfarande i en undersökande fas. Ambitionen är att hitta så bra lösningar som möjligt för att skapa samhällsnytta.

Du kan ställa en fråga eller komma med förbättringsförslag genom att använda vårt formulär ”Kontakta oss för samarbeten”. Vi vill gärna ha en öppen dialog om Skatteverkets API Rättsliga regler och konceptet med regelfiler i sin helhet.
