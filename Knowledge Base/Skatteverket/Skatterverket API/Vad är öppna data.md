## Vad är öppna data?

Öppna data är digitaliserad, offentlig information som vem som helst kan ta del av och använda till valfritt ändamål. Många svenska myndigheter tillhandahåller öppna data, bland annat Skatteverket.

Myndigheter samlar ofta in och lagrar stora mängder offentlig data. Genom att digitalisera, strukturera, och göra informationen lätt åtkomlig för andra kan den bidra till samhällsnytta även utanför myndigheten – till exempel i form av forskning eller nya produkter.

Data som görs tillgänglig på detta vis kallas ”öppna data”.

Hos Skatteverket finns offentlig data om allt från medelinkomster i olika branscher till genomsnittshyror för butikslokaler och mycket annat. Denna information arbetar vi med att göra tillgänglig för alla att komma åt och använda i form av öppna data.

## Öppna data via API

Det vanligaste sättet att komma åt öppna data är via ett så kallat API. Ett API är en funktion som en programmerare kan anropa för att ”beställa hem” information från någon annans databas eller program. Datan kan skickas till exempelvis via en app, databas eller Excelfil som tar emot och sparar en kopia av informationen. Vi kallar de öppna data som går att nå via API för öppna data-API:er (öppet API).

## Öppna data utan API

För dig som inte vill eller kan använda ett API finns ofta andra sätt att komma åt öppna data. Vissa organisationer och myndigheter publicerar öppna data i textform på sina webbplatser, andra erbjuder nedladdningsbara filer som ett komplement till API:er.

På Skatteverket tillhandahåller vi öppna data via både API:er och i form av nedladdningsbara.csv-filer (som går att öppna i Excel och andra kalkylprogram).
