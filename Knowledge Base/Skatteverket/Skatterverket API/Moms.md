Moms (mervärdesskatt) är en skatt som ingår i priset på de allra flesta varor och tjänster som vi köper. Momsen deklareras och betalas oftast av den som säljer varan eller tjänsten. Just nu utvecklar Skatteverket ett API för att underlätta redovisningen av moms.

## Kort om utvecklingsområdet

Skatteverket bedriver ett kontinuerligt utvecklingsarbete inom momsområdet på både kort och lång sikt. Tjänsteutvecklingen görs tillsammans med externa parter för att underlätta momsredovisningen. Att samarbeta med programvaruföretag, ombud eller bokförings- eller redovisningsbyråer är en förutsättning för att det API vi tar fram gör uppgiftslämnandet enklare.

Du hittar API Momsdeklaration som komplett testtjänst och driftsatt tjänst i Utvecklarportalen.

- [Momsdeklaration Länk till annan webbplats.](https://www7.skatteverket.se/portal/apier-och-oppna-data/utvecklarportalen?q=momsdeklaration)
