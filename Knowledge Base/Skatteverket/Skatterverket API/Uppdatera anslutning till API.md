Fyll i det här formuläret om du vill ändra din anslutning till ett befintligt API, till exempel för att ändra en redirect eller byta applikationsnamn.

#### Om du ska ange en organisationslegitimation.

Fyll i vilken organisationslegitimation du vill använda genom att kopiera raden "subject" från p 12-filen från certifikatutfärdaren (Expisoft). Klistra in den i fritextfältet:"Vad vill du uppdatera?". Exempel på subject-rad är: Subject: CN=Organisation A + SERIALNUMBER=165560000000, O=Organisation A, C=SE

Uppgifter markerade med \* är obligatoriska.
