## API-konsolen – så kan du använda den

API-konsolen är ett verktyg där du kan få mer information om ett enskilt API och komma igång med att utveckla tjänster som anropar våra partner-API:er.

API:ets definition berättar i mer detalj hur det fungerar praktiskt. Du kan till exempel se dokumentation om

- vilken typ av anrop som ingår
- vilka url:er du ska använda för att anropa API:et
- vilka parametrar som behövs
- vilka felkoder som finns och vad de kan bero på.

Med datan som presenteras i API-konsolen kan du som utvecklare snabbt komma igång med att implementera Skatteverkets partner-API:er i dina tjänster.

Du hittar till API-konsolen under respektive partner-API på Utvecklarportalen. Klicka på knappen API-definition under Komplett testtjänst-fliken.
