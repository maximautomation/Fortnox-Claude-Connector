## Viktig information om digitala underskrifter i Skatteverkets tjänster

Ditt avtal är giltigt– felmeddelande från din programvara beror på certifikatfel. Du kan slippa varningen genom att utöka din programvara med betrodda certifikat.

När du skrivit på ett avtal digitalt från Skatteverket behöver du ha ett betrott certifikat i den programvara du använder för att öppna avtalet. Du kommer fortfarande kunna signera avtalet utan certifikatet.

När du öppnar ditt signerade avtal utan betrott certifikat kommer du se ett felmeddelande som säger att ”Minst en signatur är felaktig”.

Ditt avtal är giltigt trots att du får detta felmeddelande.

## Lägg till certifikat i din programvara

För att slippa liknande felmeddelanden i fortsättningen kan du utöka listan över betrodda certifikat i den programvara du använder. Kontakta din IT-avdelning eller din programvaruleverantör för mer information om hur du gör.

Nödvändiga certifikat finns på sidan Produktionscertifikat hos CGI. Välj certifikaten under rubriken 'PROD - Substantial'.

- [CGI – Production Certificates Länk till annan webbplats.](https://crl.signatureservice.se/rootCA/)

Det finns ingen officiell tillitsförteckning för avancerade elektroniska signaturer för användare i Sverige. Det gör att många PDF- och webbläsare inte godkänner certifikaten som Skatteverkets leverantör CGI använder och därför varnar när certifikatet saknas.
