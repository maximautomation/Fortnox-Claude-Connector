För att få tillgång till API:et behöver du fylla i och skicka in ett formulär till Skatteverket.

För att ansöka om anslutning till tjänsten behöver kommunen fylla i kontaktformuläret. Skatteverket kontaktar sedan kommunen för vidare dialog om anslutningen.

Innan anslutningen behöver Skatteverket och kommunen göra en överenskommelse som bland annat innehåller uppgift om kommunikationssätt, förändringar, ansvar och giltighet.

När överenskommelsen är underskrivet av bägge parter får du API-nycklar i form av “client\_id” och “client\_secret”. Dessa behöver du för att anropa API:et från tjänsten. Vi kommer att skicka API-nycklar via mejl och sms. Det är därför viktigt att du skriver en korrekt e-postadress och ett korrekt mobiltelefonnummer.

Uppgifter markerade med \* är obligatoriska
