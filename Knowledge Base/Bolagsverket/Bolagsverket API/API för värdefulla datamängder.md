Vårt API för värdefulla datamängder ger dig tillgång till data som hämtas från oss på Bolagsverket och från Statistikmyndigheten SCB. Det kostar ingenting och det krävs inga avtal. Vi hoppas att det gör stor nytta för dig och din verksamhet!

Läs mer om [värdefulla datamängder](https://bolagsverket.se/apierochoppnadata/hamtaforetagsinformation/vardefulladatamangder.5294.html).

## Få tillgång till värdefulla datamängder!

Nu kan du ta del av värdefull data från Bolagsverket och SCB, avgiftsfritt och utan avtal.

Du hittar SNI -koder, digitalt inlämnade årsredovisningar och data om företag.

## Kostnaden för att använda vår tjänst

Värdefulla datamängder är avgiftsfritt. Enligt EU -kommissionens direktiv ska det vara kostnadsfritt för alla att använda. Du kan läsa mer om [värdefulla datamängder](https://bolagsverket.se/apierochoppnadata/hamtaforetagsinformation/vardefulladatamangder.5294.html).

## Gör så här för att få tillgång till API:et

### Begär tillgång

- Fyll i och skicka in [kundanmälan](https://bolagsverket.se/apierochoppnadata/hamtaforetagsinformation/vardefulladatamangder/kundanmalantillapiforvardefulladatamangder.5528.html).

### Få API-nycklar (client\_id och client\_secret) till test- och produktionsmiljö

- När vi mottagit din begäran om tillgång får du så kallade API-nycklar till test- och produktionsmiljön skickade till dig via mejl och sms enligt de kontaktuppgifter du lämnade i formuläret.

### Koppla upp och använd vår data!

- I testmiljön kan du säkerställa att dina applikationer fungerar som de ska sedan är det bara att koppla upp dig mot produktionsmiljön
- Nu kan du använda vårt API i dina system och i din verksamhet!

## Vilken information kan jag få via API:et?

Värdefulla datamängder är data som EU identifierat som särskilt viktiga för samhället. De utgör en del av den data som Bolagsverket har om företag och företagande. Vill du ha mer djupgående data så väljer du vårt API för Företagsinformation.

API för värdefulla informationsmängder hämtar data från Bolagsverket och SCB. Här ser du vilka organisationsformer (Bolagsverket) och vilka juridiska former (SCB) som datat omfattar.

- digitalt inlämnade årsredovisningar.

### Information om Dokument

I de värdefulla datamängderna ingår dokument i form av digitalt inlämnade årsredovisningar. Det finns idag inget tvång för bolag att lämna in årsredovisningen digitalt men Bolagsverket stödjer digital inlämning av aktiebolags års- och koncernredovisning enligt regelverken nedan och det är dessa du får tillgång till att hämta:

- Årsredovisning för aktiebolag enligt K2
- Årsredovisning för aktiebolag enligt K3
- Koncernredovisning för aktiebolag enligt K3
- Års- och koncernredovisning upprättade i enlighet med ESMAs ESEF-taxonomi

Genom att skicka in ett dokumentnummer (dokumentid) till API:et får du tillbaka en så kallad zip-fil med det aktuella dokumentet. Dokumentid hämtas med operationen Dokumentlista som beskrivs i listan under rubriken Information om Dokumentlista.

## Teknik

Vårt API är en REST-tjänst baserad på http och json. Anrop mot tjänsten ska krypteras via https. Auktorisation av API-anrop görs med hjälp av OAuth 2.

Prestandan för detta API tillåter varje användare att göra 60 frågor/minut.

Mer detaljerad beskrivning av vårt API hittar du i vår [utvecklarportal Länk till annan webbplats.](https://portal.api.bolagsverket.se/devportal/apis/ff8f9a91-1fdd-4705-8836-c1906581162f/overview) som finns i verktyget WSO2.
