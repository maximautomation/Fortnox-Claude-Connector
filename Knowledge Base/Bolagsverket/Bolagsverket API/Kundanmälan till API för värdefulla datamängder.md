Vårt API för värdefulla datamängder är till för dig som, utan avgift, vill ha tillgång till data från Bolagsverket och Statistikmyndigheten (SCB) och som anses extra värdefulla enligt EU-kommissionen.

Läs mer om [värdefulla datamängder](https://bolagsverket.se/apierochoppnadata/hamtaforetagsinformation/vardefulladatamangder.5294.html).

## Hög belastning på API-supporten

Vi har just nu en hög efterfrågan på våra API-tjänster och ett ökat inflöde av nya anslutningar. Det innebär att svars- och handläggningstider under en period kan vara längre än vanligt. Vi arbetar löpande för att hantera ärendemängden och besvara alla förfrågningar så snabbt som möjligt.
