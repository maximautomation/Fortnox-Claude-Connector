## Bolagsverkets och SCB:s värdefulla datamängder

Bolagsverket spelar en viktig roll i att tillhandahålla värdefulla företagsdata som nu blir tillgängliga helt avgiftsfritt. Tillsammans med Statistikmyndigheten SCB ansvarar vi för de värdefulla datamängder som ger dig tillgång till information som till exempel företagsnamn, adresser, räkenskaper och årsredovisningar. Detta gör det möjligt för dig att bygga nya digitala tjänster, ta bättre affärsbeslut och effektivisera din verksamhet. Det är snabbt och enkelt att komma igång – värdefulla datamängder är avgiftsfritt och kräver inget avtal!

Om du behöver mer djupgående data erbjuder vi avgiftsbelagda API:er med ännu mer detaljerad företagsinformation, vilket kan ge dig fler insikter och öppna upp för nya affärsmöjligheter.

## Tekniken bakom avgiftsfria data

Vi använder ett REST-API som det maskinläsbara formatet för att tillgängliggöra den värdefulla datamängden. Nedan listas de värdefulla datamängder som vi tillsammans med Statistikmyndigheten SCB tillhandahåller avgiftsfritt. Generellt handlar det om värdefull information om de organisationer som är registrerade hos oss och SCB, alla digitalt inlämnade årsredovisningar samt möjligheten att ladda ner en fil med de värdefulla datamängderna för alla organisationer.

## Påverkan på andra API:er och tjänster

EU-kommissionens beslut påverkar inte avgifter som idag finns vid användande av andra API:er och tjänster där information om företag och företagande förmedlas från oss.

Varför ska du ansluta dig?

Genom att ansluta dig till API:et för värdefulla datamängder får du tillgång till en omfattande databas med företagsinformation, helt avgiftsfritt och inget krav på avtal. I API:et finns även data från Statistikmyndigheten SCB. Det är enkelt att komma igång och data kan användas för att skapa nya digitala tjänster, göra bättre affärsbeslut eller analysera marknaden. Dessutom, om ditt behov av mer avancerad data ökar, kan du smidigt ansluta dig till våra betal-API och få ännu mer detaljerad information.

Anslut enkelt och snabbt utan avtal efter 3 februari 2025 och få tillgång till företagsinformation, räkenskaper, årsredovisningar och mycket mer.

Datamängder som ingår

- Organisationsnamn
- Organisationsform
- Reklamspärr (SCB)
- Avregistrerad organisation
- Avregistreringsorsak
- Pågående avvecklings- eller omstruktureringsförfarande
- Verksam Organisation (F-skatt, momsreg, arbetsgivreg) (SCB)
- Samtliga organisationsnamn (verksamhetsbeskrivning för särskilda företagsnamn)
- Juridisk form (SCB)
- Postadress organisation
- Registrerad organisation (registreringsdatum)
- SNI-koder organisation (SCB)
- Verksamhetsbeskrivning
- Identitetsbeteckning
- Övriga organisationsformer som inte finns hos Bolagsverket (SCB)
- Digitalt inlämnade årsredovisningar

## Vad är värdefulla datamängder?

Värdefulla datamängder är stora mängder data som EU har identifierat som särskilt viktiga, eftersom de kan bidra till innovation, förbättra offentliga tjänster och öka transparensen i samhället. Dessa datamängder, som kan innehålla allt från väderdata till företagsinformation och energiförbrukning, har en hög samhällsnytta och kan användas för att skapa digitala tjänster och applikationer som underlättar för både företag och privatpersoner. Data som tidigare krävde en avgift kommer nu vara tillgänglig utan kostnad i enlighet med EU-kommissionens direktiv.

## Varför är de värdefulla?

Värdefulla datamängder är nyckelresurser för att driva ekonomisk tillväxt och samhällelig utveckling. Genom att tillgängliggöra dessa data kan nya företag skapas, och existerande företag kan utveckla nya tekniker och tjänster som förbättrar våra liv.

Användning av värdefulla data

När det gäller värdefulla datamängder är de öppna för vidareutnyttjande enligt vissa regler. Du får använda dessa data fritt för kommersiella och icke-kommersiella syften, exempelvis för att skapa nya tjänster eller produkter, så länge användningen inte bryter mot lagar om skydd av personuppgifter eller sekretess. Data kan modifieras, bearbetas och kombineras med andra källor, vilket gör det möjligt att utveckla innovativa applikationer eller utföra djupgående analyser.

Det är dock viktigt att se till att data hanteras enligt de villkor och licenser som gäller, vilket ibland kan inkludera krav på att ange källan och säkerställa att informationen är korrekt återgiven. Digg, som ansvarar för vägledning kring öppna data, rekommenderar att information om licenser och säkerhet granskas noggrant för varje datamängd.

Mer information om reglerna kring vidareutnyttjande av värdefulla datamängder kan du hitta på sidan [Öppna datadirektivet Länk till annan webbplats.](https://www.digg.se/kunskap-och-stod/eu-rattsakter/oppna-datadirektivet) på Diggs webbplats.

## Hur används dessa datamängder?

EU:s mål med att göra dessa datamängder tillgängliga är att de ska kunna användas av forskare, företag och offentliga sektorn för att

- förbättra beslutsfattande och policyutveckling
- stimulera innovation genom att stödja utvecklingen av nya applikationer och tjänster
- öka effektiviteten i offentliga tjänster.

## EU:s roller och regler

EU arbetar för att säkerställa att dessa värdefulla datamängder är tillgängliga på ett sätt som skyddar individens integritet men samtidigt främjar öppen tillgång och användning av datan. Detta innefattar att fastställa regler för hur datan får hanteras och vem som har tillgång till den.

## Avgiftsfri data

Från den 3 februari 2025 blev Bolagsverkets och Statistikmyndigheten SCB:s värdefulla data avgiftsfria. Förordningen trädde i kraft den 9 februari 2023 och regeringen beviljades ett undantag för flera myndigheter däribland Bolagsverket och SCB, vilket innebär en tvåårig övergångsperiod innan avgiftsfriheten träder i kraft.

## Öppna data väntas ge stora ekonomiska samhällsvinster

Samhällsnyttan av fria offentliga data har värderats till 10–21 miljarder kronor per år bara i Sverige. Det är dessa särskilt värdefulla data som ska finnas som öppna data utan avgifter. Det har EU-kommissionen nu beslutat. Tidigare avgiftsbelagda data hos Bolagsverket och Statistikmyndigheten SCB om företag, till exempel adresser och årsredovisningar blir därmed avgiftsfria om de hämtas via ett API-gränssnitt eller via en avsedd fil.
