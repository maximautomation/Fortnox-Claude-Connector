## Anslut till den Sammansatta bastjänsten för grunddata om organisationer (SSBTGO)

Så här ansluter ni som offentlig aktör till den Sammansatta bastjänsten för grunddata om organisationer (SSBTGO).

Skriv ut och skriv under anslutningsavtalet.
