Här kan du hämta aktuell företagsinformation från Bolagsverket, via bland annat API:er. Du får tillgång till uppdaterad information som kan användas i allt från affärsutveckling till automatiserade kontroller och direkt i dina system om du väljer API.

Här är en lista på de tjänster du kan ansluta dig till för att hämta företagsinformation från Bolagsverket.

[API för att hämta företagsinformation](https://bolagsverket.se/apierochoppnadata/hamtaforetagsinformation/apiforatthamtaforetagsinformation.3988.html) För företag, privatpersoner och offentlig verksamhet.

[API för värdefulla datamängder](https://bolagsverket.se/apierochoppnadata/hamtaforetagsinformation/vardefulladatamangder/apiforvardefulladatamangder.5513.html) För företag och privatpersoner.

[Hämta information om verklig huvudman](https://bolagsverket.se/apierochoppnadata/hamtaforetagsinformation/hamtainformationomverklighuvudman.6009.html) För myndigheter och verksamhetsutövare.

[API för att hämta grunddata om organisationer (SSBTGO)](https://bolagsverket.se/apierochoppnadata/hamtaforetagsinformation/apiforatthamtagrunddataomorganisationerssbtgo.2227.html) För offentlig verksamhet.

[Nedladdningsbara filer](https://bolagsverket.se/apierochoppnadata/hamtaforetagsinformation/nedladdningsbarafiler.2517.html) För företag och privatpersoner.

[Företagsinformation via XML](https://bolagsverket.se/apierochoppnadata/hamtaforetagsinformation/foretagsinformationviaxml.2017.html) För företag och privatpersoner.

[Avisering](https://bolagsverket.se/apierochoppnadata/hamtaforetagsinformation/avisering.2009.html) För företag, privatpersoner och offentlig verksamhet

## API för att hämta företagsinformation

Få tillgång till aktuell företagsinformation, direkt i dina egna system. Med vårt API kan du hämta uppgifter som namn, status, adress och registreringsdatum snabbt, säkert och automatiserat. Perfekt för allt från kundkännedom till smarta tjänster.
