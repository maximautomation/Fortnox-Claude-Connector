## Lämna företagsinformation

Med våra API:er kan du skicka in information till Bolagsverket direkt från dina egna system. Just nu kan du skicka in styrelseändringar i bostadsrättsföreningar och anmäla kungörelser till Bolagsverket. Vi erbjuder också en tjänst för digital inlämning av årsredovisning och revisionsberättelse för aktiebolag.

## Öppna data

Öppna data är information som alla får använda, återanvända och dela. På Bolagsverket erbjuder vi flera typer av öppna data – vissa är avgiftsfria, andra kräver avtal.

Data finns tillgänglig via API:er och som nedladdningsbara filer, och kan användas för utveckling, analys och insyn.

### Exempel på vår öppna data

- företagsstatistik
- digitalt inlämnade årsredovisningar
- grundläggande företagsinformation.
