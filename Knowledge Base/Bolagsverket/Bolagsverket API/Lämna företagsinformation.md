Vill du skicka in uppgifter till Bolagsverket direkt från dina egna system? Med våra API:er kan du lämna information digitalt – till exempel styrelseändringar eller kungörelser. Det sparar tid och minskar risken för fel.

Här är en lista på de tjänster du kan ansluta dig till för att lämna företagsinformation till Bolagsverket.

[API för styrelseändringar i bostadsrättsföreningar](https://bolagsverket.se/apierochoppnadata/lamnaforetagsinformation/apiforstyrelseandringaribostadsrattsforeningar.2533.html) För bostadsrättsföreningar.

[API för att registrera kungörelser](https://bolagsverket.se/apierochoppnadata/lamnaforetagsinformation/apiforattregistrerakungorelser.2535.html) För myndigheter och kommuner.

[Tjänst för digital inlämning av årsredovisning och revisionsberättelse](https://bolagsverket.se/apierochoppnadata/lamnaforetagsinformation/digitalinlamningavarsredovisningochrevisionsberattelse.5935.html) För företag som erbjuder programvara eller system för användare.
