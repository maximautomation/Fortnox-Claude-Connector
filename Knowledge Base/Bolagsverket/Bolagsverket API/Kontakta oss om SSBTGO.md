## Kontakta oss om den Sammansatta bastjänsten för grunddata om organisationer (SSBTGO)

Vill du veta mer om hur vi tillsammans och med den Sammansatta bastjänsten för grunddata om organisationer (SSBTGO) kan underlätta för företag samt för offentliga aktörer? Kontakta oss så berättar vi mer.

## Frågor om anslutning, avtal och allmänt om tjänsten
