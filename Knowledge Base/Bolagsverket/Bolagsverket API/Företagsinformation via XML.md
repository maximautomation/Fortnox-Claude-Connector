Vill du hämta uppgifter direkt från näringslivsregistret och använda ett eget gränssnitt, då kan du använda våra XML-paket. Som kund får du alltid tillgång till nyregistrerade händelser i registret.

## Förändringar för XML-paket under 2026

Vi moderniserar våra tjänster för att hämta företagsinformation. Det innebär att vi succesivt kommer stänga våra XML-paket. Du som idag använder dessa behöver därför börja planera och övergå till det nya API:et för att hämta företagsinformation.

## Vilken information kan jag få via XML-paket?

Företagsinformation via XML ger dig den aktuella informationen som finns i näringslivsregistret. Informationen erbjuds i olika XML-paket där man som användare väljer det eller de paket som passar de behov man har. Som kund har du alltid tillgång till nyregistrerade händelser i registret. Läs mer om våra XML-paket under rubriken Produktpaket.

Tjänsten innebär att du hämtar uppgifter direkt från näringslivsregistret via så kallade webbserviceanrop, med svar direkt till ett eget gränssnitt eller integrerat direkt i era verksamhetssystem (till exempel som förifylld information).

Som XML-kund får du tillgång till Bolagsverkets näringslivsregister som bland annat innehåller aktiebolags-, handels- och föreningsregistren där du kan hämta information om styrelser, årsredovisningar och olika företags aktiekapital.

För att kunna använda tjänsten behöver du ett klientcertifikat. Det beställer du hos Expisoft i form av en serverlegitimation.

På sidan [Villkor för försäljning](https://bolagsverket.se/apierochoppnadata/hamtaforetagsinformation/foretagsinformationviaxml/villkorvidanvandandetavxmlprodukter.2025.html) kan du läsa mer om vad som gäller för dig som kund för att använda våra XML-paket. Under rubriken Kostnaden för att använda vår tjänst på den här sidan ser du vad det kostar.

All grunddata som registreras på Bolagsverket samlas i näringslivsregistret. När du söker information i näringslivsregistret hämtas och presenteras uppgifter ur dessa register:

- Aktiebolagsregistret
- Handelsregistret
- Bankregistret
- Försäkringsregistret
- Försäkringsförmedlarregistret
- Registret över grupperingar för territoriellt samarbete (EGTS -registret)
- Filialregistret
- Näringsförbudsregistret
- Biträdesförbudsregistret
- Konkursregistret, avser fysiska personer och dödsbon (från och med den 1 juni 2019 finns uppgifterna i ett samlat insolvensregister över konkurser)
- Registret mot penningtvätt
- Inteckningsbrevsregistret
- Registret över verkliga huvudmän (RVH)

På sidan Vad vi registrerar kan du läsa mer om vilka uppgifter vi registrerar. Vi får ofta frågor där vi måste hänvisa vidare till andra myndigheter och organisationer. Om du är osäker på vart du ska vända dig kan du läsa om vad vi inte registrerar på sidan Vad vi inte registrerar.
