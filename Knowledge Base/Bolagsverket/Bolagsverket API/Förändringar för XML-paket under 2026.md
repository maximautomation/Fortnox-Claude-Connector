Vi moderniserar våra tjänster för att hämta företagsinformation. Det innebär att vi succesivt kommer stänga våra XML-paket. Du som idag använder dessa behöver därför börja planera och övergå till det nya API:et för att hämta företagsinformation.

## Vad händer nu?

Under 2026 händer följande:

- Från och med den 1 maj 2026 kommer vi inte längre att ansluta nya kunder till våra XML-paket.
- Vi planerar att stänga våra XML-paket senast den 30 juni 2028.

Du kan redan idag ansöka om att få tillgång till en testmiljö där du i lugn och ro kan bekanta dig med det nya API:et – API för att hämta företagsinformation.

## Migrera från XML till API – steg för steg

1. Identifiera vilka tjänster hos er som idag använder våra XML-paket.
2. Fundera på vilken information ni har behov av och hur många transaktioner ni använder varje månad.
3. Läs igenom informationen om vårt nya [API för att hämta företagsinformation](https://bolagsverket.se/apierochoppnadata/hamtaforetagsinformation/apiforatthamtaforetagsinformation.3988.html). Där finns information om avtal, priser, tekniska beskrivningar och hur ni får tillgång till testmiljön.
4. Fyll i [kundanmälan](https://bolagsverket.se/apierochoppnadata/hamtaforetagsinformation/apiforatthamtaforetagsinformation/kundanmalantillapiforatthamtaforetagsinformation.4708.html) och få tillgång till testmiljön för API:et.
5. Börja testa och utforska möjligheterna med vårt nya API!

## Bakgrund

Vi har länge erbjudit XML-paket för att hämta uppgifter direkt från Näringslivsregistret för att sen använda informationen i egna gränssnitt och lösningar.

Sedan 2022 har vi jobbat med att utveckla ett nytt API för att förmedla företagsinformation, som går från att använda SOAP till REST-teknik. Den nya versionen har varit tillgänglig sedan 2022 men har inte innehållit samma information som XML-paketet. Under slutet av 2026 kommer innehållet att vara likadant.
