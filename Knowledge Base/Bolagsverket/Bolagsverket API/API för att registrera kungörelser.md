Vi erbjuder ett API för att registrera kungörelser i Post- och Inrikes Tidningar (PoIT).

[Post- och Inrikes Tidningar (PoIT) Länk till annan webbplats.](https://poit.bolagsverket.se/poit-app/)är Sveriges officiella kungörelseorgan. PoIT innehåller kungörelser från myndigheter och andra som måste kungöra i PoIT för att information ska komma till allmänhetens kännedom.

Vi erbjuder ett API för att registrera kungörelser i PoIT. Vi har däremot inget API för att läsa och söka kungörelser.
