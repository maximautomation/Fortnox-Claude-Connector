# Fortnox AI-integrationer - ChatGPT Connector och Claude Connector

## Översikt

Detta dokument beskriver två AI-integrationer för Fortnox:

- ChatGPT Connector
- Claude Connector

Båda integrationerna kopplar Fortnox till en AI-assistent så att användaren kan hantera bokföringsrelaterade arbetsflöden genom att beskriva vad som ska göras i klartext.

Båda integrationerna är byggda av Sidcom AB.

## Gemensam grundfunktion

Både ChatGPT Connector och Claude Connector gör det möjligt att koppla Fortnox till en AI-assistent och arbeta med Fortnox-data i klartext.

Exempel på arbetsflöden som nämns:

- hantera reskontra
- hantera fakturor
- hantera bokföring
- skapa och granska rapporter
- skapa kunder
- skapa fakturor
- skapa verifikat
- sammanställa perioder
- jämföra rapportperioder
- kontrollera vad som behöver stämmas av inför momsdeklaration

Exempel på frågor eller kommandon:

- Vilka kunder har förfallna fakturor över 30 dagar?
- Bokför kvittot jag laddade upp.
- Sammanställ Q1 och jämför med Q1 i fjol.
- Vad behöver jag stämma av inför momsdeklarationen?

### Beskrivning

ChatGPT Connector kopplar Fortnox till ChatGPT.

Integrationens syfte är att användaren ska kunna hantera bokföring i klartext istället för att klicka genom menyer.

Användaren kan beskriva vad som ska göras i ChatGPT.

ChatGPT föreslår åtgärder och användaren bestämmer.

Arbetsflödet sker i klartext direkt i ChatGPT på webben eller desktop.

### Exempel på användning

ChatGPT Connector kan användas för att fråga, skapa och granska bokföring direkt i ChatGPT.

Exempel från källtexten:

- Vilka kunder har förfallna fakturor över 30 dagar?
- Bokför kvittot jag laddade upp.
- Sammanställ Q1 och jämför med Q1 i fjol.
- Vad behöver jag stämma av inför momsdeklarationen?

ChatGPT Connector kan användas för att skapa:

- kunder
- fakturor
- verifikat

### Kontroll och godkännande

ChatGPT föreslår alltid.

ChatGPT ber om användarens godkännande innan något skickas eller ändras.

Användaren behåller kontrollen.

### Stöd för Fortnox-arbetsflöden

ChatGPT Connector anges fungera för Fortnox-vardagen inom:

- reskontra
- fakturor
- leverantörer
- rapporter
- moms

### Så fungerar kopplingen med Fortnox

Företaget aktiverar ChatGPT Connector i Fortnox-portalen en gång.

Efter aktivering lägger varje användare till kopplingen som en app i ChatGPT.

Användaren loggar in via Fortnox OAuth.

Fortnox sköter inloggningen.

Inga lösenord lagras hos Sidcom.

Användarens egna Fortnox-behörigheter styr vad ChatGPT får göra.

### Data och lagring

Din bokföring lagras endast i Fortnox.

Sidcom sparar ingen bokföringsdata.

Chattinnehåll behandlas av OpenAI.

### Plattformar

ChatGPT Connector fungerar i:

- ChatGPT på webben
- ChatGPT desktop
- Codex

ChatGPT Connector kräver ett betalt ChatGPT-abonnemang.

### Beskrivning

Claude Connector kopplar Fortnox till Claude.

Integrationens syfte är att användaren ska kunna hantera hela bokföringen i klartext istället för att klicka genom menyer.

Användaren kan beskriva vad som ska göras i Claude.

Claude föreslår innan något skickas eller ändras.

Användaren behåller kontrollen.

Arbetsflödet sker i klartext direkt från Claude Desktop, Claude Code eller mobilen.

### Exempel på användning

Claude Connector kan användas för att fråga, skapa och granska bokföring direkt i Claude.

Exempel från källtexten:

- Vilka kunder har förfallna fakturor över 30 dagar?
- Bokför kvittot jag laddade upp.
- Sammanställ Q1 och jämför med Q1 i fjol.
- Vad behöver jag stämma av inför momsdeklarationen?

Claude Connector kan användas för att skapa:

- kunder
- fakturor
- verifikat

### Kontroll och godkännande

Claude föreslår alltid innan något skickas eller ändras.

Skrivåtgärder kräver alltid bekräftelse innan något skickas eller ändras.

Användaren behåller kontrollen.

### Så fungerar kopplingen med Fortnox

Företaget aktiverar Claude Connector i Fortnox-portalen en gång.

Efter aktivering ansluter varje användare sin Claude.

Användaren kan ansluta via:

- Claude Desktop
- Claude mobil
- Claude Code

Användaren loggar in via Fortnox OAuth.

Fortnox sköter inloggningen.

Inga lösenord lagras hos Sidcom.

Användarens egna Fortnox-behörigheter styr vad Claude får göra.

### Revisionslogg

Claude Connector har full revisionslogg.

Användaren ser varje anrop, från vem och när.

### Plattformar

Claude Connector fungerar med:

- Claude Desktop
- Claude Code
- Claude mobil
